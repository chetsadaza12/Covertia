# Blocks Group Workflow

ตัวอย่าง Workflow สาธิตการทำงานของบล็อก **Blocks Group** ซึ่งเป็นคำสั่งควบคุมประเภทโครงสร้างและโฟลเดอร์สำหรับยุบรวม บันทึกกลุ่ม หรือคัดแยกกลุ่มของบล็อกย่อยต่าง ๆ (Sub-blocks Container) เพื่อความเป็นระเบียบเรียบร้อยของหน้า Canvas และช่วยแบ่งการทำงานออกเป็นโมดูลย่อยที่สามารถเปิด/ปิดการทำงานหรือประมวลผลย่อยได้ในชุดเดียวกัน

## Interactive Workflow Preview

คลิกที่บล็อกเพื่อแก้ไขข้อมูล ลากโฮเวอร์บล็อกอื่น ๆ แล้วคลิกปุ่ม **"Move to Group"** (ไอคอนกล่องประในแถบเครื่องมือลอย) เพื่อย้ายบล็อกนั้น ๆ เข้าไปในกลุ่มได้จริง! และคลิกปุ่ม **`×`** บนรายการกลุ่มเพื่อดึงบล็อกกลับออกมาภายนอกได้

## Workflow Blocks

| Block | Type | Description |
|-------|------|-------------|
| **Start (Trigger)** | `BlockBasic` | จุดเริ่มต้นเวิร์กโฟลว์ |
| **Open URL** | `BlockBasicWithFallback` | เปิดหน้าเว็บปลายทาง |
| **Blocks Group** | `BlockGroup` | คอนเทนเนอร์รวมโหนดต่าง ๆ (สามารถนำโหนดอื่นย้ายเข้ามาจัดเก็บได้) |
| **Browser Request** | `BlockBasicWithFallback` | บล็อกสำหรับการเชื่อมต่อส่งคำขอ HTTP Request (ในตัวอย่างด้านบนจำลองวางไว้ด้านนอกเพื่อให้คุณทดลองคลิกย้ายเข้ากลุ่ม) |

## ฟีเจอร์และการใช้งาน Blocks Group

บล็อก **Blocks Group** ทำงานเสมือนโฟลเดอร์รวบรวมไฟล์ ช่วยลดความหนาแน่นของแผนภาพเวิร์กโฟลว์ขนาดใหญ่

### ค่าพารามิเตอร์

| พารามิเตอร์ | คำอธิบาย | ตัวเลือก / รายละเอียดเพิ่มเติม |
| :--- | :--- | :--- |
| **Group name** | ชื่อสำหรับระบุกลุ่มของบล็อก | ป้อนชื่อระบุ เช่น `ดึงข้อมูลโปรไฟล์`, `ล็อกอินสับเปลี่ยนบัญชี` |
| **Nested Blocks** | บล็อกย่อยที่อยู่ภายในกลุ่ม | แสดงเป็นรายการแถบสีเทาด้านในพร้อมไอคอน โดยจะประมวลผลแบบเป็นลำดับขั้นภายในกลุ่ม |

### ขั้นตอนการย้ายบล็อกเข้า/ออกกลุ่ม (Interactive Control)

*   **วิธีการย้ายเข้ากลุ่ม (Move-In)**:
    1.  คลิกหรือโฮเวอร์ไปที่โหนดบนหน้าจอที่ต้องการย้าย (เช่น **Browser Request**)
    2.  จะปรากฏแถบเครื่องมือด่วนด้านบนไอคอนรูปถังขยะ ให้คลิกที่ไอคอน **"Move to Group"** (รูปสี่เหลี่ยมกรอบประพร้อมหัวลูกศรชี้ 📥)
    3.  บล็อกจะถูกย้ายเข้าไปบรรจุเป็นแถบสีเทาในกลุ่ม **Blocks Group** โดยอัตโนมัติ และขนาดความสูงของกลุ่มจะยืดขยายตามจำนวนบล็อกจริง
*   **วิธีการดึงออกนอกกลุ่ม (Move-Out)**:
    1.  ในรายชื่อบล็อกสีเทาด้านใน **Blocks Group**
    2.  กดเครื่องหมายกากบาท **`×`** ที่มุมขวาของรายการ ระบบจะดึงบล็อกย่อยนั้นกลับออกมาวางบนพื้นที่ระนาบหลักของหน้าจอ Canvas ในตำแหน่งด้านล่างกลุ่มทันที
