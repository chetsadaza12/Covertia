# Keyboard Shortcuts

Using keyboard shortcuts allows you to navigate the application and execute common actions or workflow commands quickly.

---

## Core System Shortcuts

Global core system hotkeys:

| Action | Keyboard Shortcut |
| :--- | :--- |
| **Shortcut Panel** | `Ctrl + Shift + E` |

---

## Page Navigation Shortcuts

Quickly jump between different dashboards and modules using these hotkeys:

| Target Page | Keyboard Shortcut |
| :--- | :--- |
| **Profile** | `Alt + 1` |
| **Workflows** | `Alt + 2` |
| **Marketplace** | `Alt + 3` |
| **Package** | `Alt + 4` |
| **Extension** | `Alt + 5` |
| **Gem Ecosystem** | `Alt + G` |
| **Schedule** | `Alt + 6` |
| **Storage** | `Alt + 7` |
| **Logs** | `Alt + 8` |
| **Api** | `Alt + 9` |
| **Settings** | `Alt + 0` |

---

## Common Action Shortcuts

Execute global search or record creation commands:

| Action | Keyboard Shortcut |
| :--- | :--- |
| **Search** | `Ctrl + F` |
| **New Record** | `Ctrl + Alt + N` |

---

## Workflow Editor Shortcuts

These hotkeys are active when editing a workflow script in the visual canvas workspace:

| Editor Action | Keyboard Shortcut |
| :--- | :--- |
| **Duplicate Block** | `Ctrl + Alt + D` |
| **Search Blocks** | `Ctrl + B` |
| **Save Script** | `Ctrl + S` |
| **Execute Workflow** | `Alt + Enter` |
