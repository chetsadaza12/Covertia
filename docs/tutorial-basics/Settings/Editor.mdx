# Editor Settings

The **Editor Settings** panel allows you to customize the visual workflow editor canvas behavior, block connections, and canvas snapping parameters.

---

## Canvas Zoom Limits

Define the minimum and maximum zoom scale boundaries for navigating the visual canvas:

| Setting | Description | Default / Options |
| :--- | :--- | :--- |
| **Min zoom** | The minimum zoom-out multiplier scale factor for the canvas workspace. | Decimal input (e.g. `0.1` for 10% zoom) |
| **Max zoom** | The maximum zoom-in multiplier scale factor for the canvas workspace. | Decimal input (e.g. `1.3` for 130% zoom) |

---

## Editor Behavior Options

Configure how connector lines, block positioning, and script saves behave in the builder workspace:

| Option | Description | Toggle Type |
| :--- | :--- | :--- |
| **Line arrow** | Add a directional arrow head at the end of connector lines linking workflow block nodes. | Switch toggle (On/Off) |
| **Snap to the grid** | Automatically align blocks to the canvas grid system when moving or placing them. | Switch toggle (On/Off) |
| **Auto-save** | Automatically save all changes made to the workflow script layout whenever the workflow is executed. | Switch toggle (On/Off) |

---

## Save Configuration

Click the **Save** button at the bottom of the panel to persist your canvas preferences.
