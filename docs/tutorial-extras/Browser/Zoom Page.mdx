# Zoom Page Workflow

ตัวอย่าง Workflow สาธิตการทำงานของบล็อก **Zoom Page** ซึ่งเป็นคำสั่งสำหรับการปรับปรุงอัตราการซูมย่อขยายขนาดการแสดงผลของหน้าเว็บเพจ (Page Zoom Level Control) เพื่ออำนวยความสะดวกในการจับภาพหน้าจอ (Take Screenshot) หรือปรับให้องค์ประกอบของเว็บสอดคล้องกับพฤติกรรมการตรวจจับรูปภาพอัตโนมัติ

## Workflow Blocks

| Block | Type | Description |
|-------|------|-------------|
| **Start (Trigger)** | `BlockBasic` | จุดเริ่มต้นทำงานเวิร์กโฟลว์ |
| **Open URL** | `BlockBasicWithFallback` | เปิดหน้าเบราว์เซอร์เป้าหมาย |
| **Zoom Page** | `BlockBasicWithFallback` | บล็อกซูมย่อหรือขยายระดับหน้าเว็บเพจตามเปอร์เซ็นต์ที่ระบุ |

## การตั้งค่าบล็อก Zoom Page

บล็อก **Zoom Page** ช่วยให้เวิร์กโฟลว์ปรับขนาดมุมมอง (Zoom Level) ของหน้าเพจปัจจุบัน

### ค่าพารามิเตอร์หลัก

| พารามิเตอร์ | คำอธิบาย | ตัวเลือก / รายละเอียดเพิ่มเติม |
| :--- | :--- | :--- |
| **Description** | คำอธิบายรายละเอียดสั้น ๆ สำหรับบล็อกนี้ | แสดงผลเป็นฟิลด์หัวเรื่องไร้กรอบด้านบนสุด |
| **Zoom size** | สัดส่วนเปอร์เซ็นต์ที่ต้องการใช้ในการซูมหน้าเว็บเพจ | ค่าเริ่มต้นคือ `100` (ซูมปกติ 100%) รองรับการใช้ตัวแปร `{x}` |

### ค่าในส่วน Settings

*   **Timeout (millisecond)**: กำหนดเวลาสูงสุดในการทำงานของบล็อกนี้
*   **Delay time (millisecond)**: เวลาหน่วงก่อนเริ่มประมวลผลคำสั่งในบล็อก

### ค่าในส่วน On error

*   **Enable Switch**: เปิด/ปิดการทำงานข้ามข้อผิดพลาดเมื่อเกิดปัญหาระหว่างคำสั่งซูมหน้าเพจ

---

### กรณีการใช้งาน (Use Cases)

*   **High-Resolution Screenshotting**: ปรับอัตราการซูมหน้าเว็บลง (เช่น ซูม `75%` หรือ `50%`) เพื่อขยายพื้นที่การแสดงผลหน้าเพจให้กว้างขวางขึ้นก่อนส่งคำสั่งเก็บภาพหน้าจออัตโนมัติ ช่วยให้เก็บรายละเอียดของหน้าเว็บที่มีเนื้อหายาว ๆ หรือกว้างได้ครบถ้วน
*   **Element Match Alignment**: ในการใช้เครื่องมือค้นหาและวิเคราะห์รูปภาพบนหน้าจอ (Image Search) การปรับซูมให้ได้สัดส่วนตรงกับชุดภาพต้นแบบจะช่วยเพิ่มอัตราความแม่นยำในการค้นหาและประมวลผลให้สูงยิ่งขึ้น
