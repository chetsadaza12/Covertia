# Advanced Features

---

## Profile Operations

These core control buttons are available on the dashboard:

* **Open**: Button used to open the browser profile.
* **Close**: Button used to close the browser profile.
* **Delete**: Button used to delete the browser profile.

---

## Sync Operation

The **Sync Operation** dialog is used to configure and start synchronized actions across multiple browser profile windows.

---

### Configuration Fields

#### Select screen
Dropdown to select the monitor/screen where the windows will open.

#### Select main profile
Dropdown to select the primary controller profile (e.g., `1: Private-Description`).

#### Profile open delay (ms)
Input field to set the delay time in milliseconds between opening each profile (Default: `300`).

#### Window scale
Input field to set the display scale multiplier (Default: `1.0`).

---

### Window size Layouts

* **Auto Split Evenly**: Automatically splits and arranges browser windows evenly across the screen.
* **Fixed Size**: Sets a fixed dimension for the windows. When selected, the following inputs are available:
  * **Width**: Default `500`
  * **Height**: Default `500`
* **Grid Layout**: Arranges windows in a grid pattern. When selected, the following inputs are available:
  * **Rows**: e.g., `2`
  * **Columns**: e.g., `3`

---

### Toggles

* **Auto open and sort**: Toggle to automatically open profiles and arrange them according to the selected window size layout.
* **Proxy**: Toggle to enable/disable proxy settings.
* **Fixed main profile**: Toggle to fix the main controller profile window position.

---

### Run Button

* **Start Sync Actions**: Button to initiate the synchronization process.

---

## Column setting

The **Column setting** dialog allows you to toggle the visibility of columns in the profile list table.

---

### Available Columns

Check or uncheck the following fields to customize the table layout:

* `#`
* `Name`
* `Data resource`
* `Group`
* `Last Interact`
* `Message`
* `ID`
* `Resource`
* `Proxy`
* `Note`
* `Status`
* `IP`
