# End

บล็อก **End** เป็นโหนดยุติการทำงานของสคริปต์เวิร์กโฟลว์ทั้งหมดโดยสมบูรณ์ โดยบล็อกนี้จะเชื่อมต่อเป็นจุดสุดท้ายของกระบวนการ Automation

## รายละเอียดบล็อก End

บล็อก **End** ทำหน้าที่เป็นจุดสิ้นสุดของกระบวนการทำงานทั้งหมด เมื่อระบบดำเนินการมาถึงบล็อกนี้ สคริปต์จะหยุดทำงานทันทีและส่งผลลัพธ์สถานะเป็นสำเร็จ (Success)

### การตั้งค่า

*   **Description**: คำอธิบายสั้น ๆ ของบล็อกสำหรับการจดบันทึกภายใน Workflow
*   *หมายเหตุ: บล็อก End เป็นบล็อกปลายทาง (Terminal Block) จึงไม่มีพอร์ตขาออก (Output Port) สำหรับลากเส้นเชื่อมโยงต่อไปยังบล็อกอื่นอีก*

---

### กรณีการนำไปใช้งาน (Use Cases)

*   **ยุติการทำงานเมื่อสำเร็จ**: วางไว้ท้ายสุดของลำดับขั้นใน Workflow เพื่อระบุว่าสคริปต์ทำงานครบขั้นตอนและสิ้นสุดกระบวนการแล้ว
*   **หยุดสคริปต์ตามเงื่อนไข (Conditional Stop)**: ใช้ร่วมกับบล็อกตรวจสอบเงื่อนไข (Conditions) หากเงื่อนไขตรงตามที่กำหนด ให้เชื่อมโยงเส้นมาที่บล็อก **End** เพื่อสั่งยุติการทำงานโดยไม่ต้องดำเนินการต่อ
*   **ตัดจบการทำงานในลูป**: เมื่อทำงานในลูปแล้วได้ผลลัพธ์ที่ต้องการเรียบร้อยแล้ว ให้สั่งวิ่งออกนอกลูปเข้าหาบล็อก **End** เพื่อจบการรันสคริปต์
